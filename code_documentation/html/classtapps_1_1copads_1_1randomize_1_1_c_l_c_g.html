def tapps.copads.randomize.CLCG.random (   self)

Method to generate a random integer using the following equation where s is a newly generated integer.

x(n+1) = [multiplierX * x(n) + incrementX] % modulusX

y(n+1) = [multiplierY * y(n) + incrementY] % modulusY

s = [x(n+1) + y(n+1)] % max(modulusX, modulusY)

Returns
a random generated integer.
